# Auth System

A simple and secure authentication system built with **Node.js**, **Express**, and **JWT**. This project allows users to register, log in, and securely manage authentication.

## Features
- User registration and login
- Password hashing with bcrypt
- JWT-based authentication for secure sessions
- Basic error handling for authentication routes
- Organized folder structure for scalability
